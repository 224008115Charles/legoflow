## 功能

* Webpack 4x 的 JS 模块编译，支持各类模块规范
* Babel 7 ESNext JS 编译
* Vue.js 开发环境
* Sass 预编译样式文件
* CSS Autoprefixer 自动补全
* 移动端 REM 方案
* Base64 图片，多张雪碧图合并
* 生成图片信息，智能图片压缩
* EJS 模板
* 内联样式脚本，替换线上静态资源地址
* JS 编译 Sass 文件 Autoprefixer 自动补全
* JS 编译 art-template 语法 tpl 文件
* ......
