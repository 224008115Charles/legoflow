## 功能

* Webpack 4x 的 JS 模块编译，支持各类模块规范
* Babel 7 ESNext JS 编译
* Vue.js 2x 开发环境
* Sass 编译
* Autoprefixer 自动补全
* 移动端 REM 适配方案
* 生成图片信息，智能图片压缩
* Base64 图片，雪碧图
* EJS 模板编译
* 自动内联样式脚本
* 自动替换线上资源地址
* JS 编译 Sass 文件 Autoprefixer 自动补全
* JS 编译 [art-template](https://aui.github.io/art-template/zh-cn/index.html) 语法 tpl 文件
* 提供工作流前后拦截器，自定义功能
* ......
